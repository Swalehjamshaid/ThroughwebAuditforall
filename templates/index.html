Web Audit Pro — Downloads

Download the Python-based Web Audit Pro tool and use the Quick Start commands below. This tool includes asynchronous crawling, executive PDF report & charts, PageSpeed Insights enrichment (mobile/desktop), Google Search Console indexed-pages estimate, and SEMrush/Ahrefs placeholders.

Download web_audit_pro.py

Place this HTML and the Python file in the same folder for offline downloads.

Quick Start — Pro (CLI)

python web_audit_pro.py \
  --url https://example.com \
  --max-pages 100 \
  --timeout 10 \
  --user-agent "WebAuditPro/1.0" \
  --respect-robots \
  --concurrency 20 \
  --pagespeed-api-key YOUR_PSI_KEY \
  --pagespeed-strategy mobile \
  --gsc-credentials path/to/creds.json \
  --gsc-property https://example.com/ \
  --semrush-key YOUR_SEMRUSH_KEY \
  --ahrefs-key YOUR_AHREFS_KEY \
  --out audit_report.json \
  --pdf-out audit_report.pdf

Dependencies

Install prerequisites:

pip install requests beautifulsoup4
pip install httpx reportlab matplotlib google-api-python-client google-auth

Notes

  • Use --pagespeed-strategy to switch between mobile and desktop PSI analysis.
  • Provide a valid Google PSI key and a GSC service account JSON for indexed-pages estimates.
  • SEMrush/Ahrefs endpoints are placeholders — plug in your plan’s API URLs & tokens.
